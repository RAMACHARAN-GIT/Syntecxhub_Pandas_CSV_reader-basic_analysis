# Pandas CSV Reader & Basic Analysis

## Project Description
**Pandas CSV Reader & Basic Analysis** is a data analysis project focused on reading, exploring, and processing CSV and Excel files using Pandas. The project demonstrates essential data inspection, filtering, and export techniques.

---

## Features & Learnings
- Read CSV and Excel files into Pandas DataFrames  
- Inspect data using head(), tail(), and data types  
- Compute summary statistics such as mean, median, minimum, maximum, and count  
- Filter rows, select columns, and slice data subsets  
- Save filtered results back to CSV and Excel formats  

---

## Technologies Used
- Python  
- Pandas
